# 📚 도서 대여 관리 시스템

Python Flask와 SQLite를 사용한 도서 대여 상태 관리 시스템입니다.

## 주요 기능

✅ 책 목록 관리 (추가/수정/삭제)
✅ 책 대여/반납 기능
✅ 대여 상태 필터링 (전체/대여가능/대여중)
✅ 책 검색 (제목, 저자)
✅ 대여 통계 대시보드
✅ 책 이미지 표시
✅ 반납 예정일 자동 계산 (14일)

## 설치 및 실행

### 1. 필요한 패키지 설치

```bash
pip install -r library_requirements.txt
```

### 2. 애플리케이션 실행

```bash
python library_app.py
```

### 3. 브라우저에서 접속

```
http://localhost:5001
```

## 데이터 구조

### books 테이블
- **id**: 책 ID (자동 증가)
- **title**: 책 제목
- **author**: 저자
- **image_url**: 책 이미지 URL
- **borrower**: 대여자 이름
- **borrow_date**: 대여일
- **return_date**: 반납 예정일
- **status**: 대여 상태 (available/borrowed)

## 사용 방법

### 책 추가하기
1. 메인 페이지에서 "+ 책 추가" 버튼 클릭
2. 책 제목, 저자 입력
3. 인터넷에서 책 이미지 검색 후 이미지 URL 복사하여 입력
4. "추가하기" 버튼 클릭

### 책 이미지 URL 찾는 방법
1. 구글 이미지 검색 또는 yes24.com, aladin.co.kr에서 책 검색
2. 책 표지 이미지에 마우스 우클릭
3. "이미지 주소 복사" 선택
4. 입력란에 붙여넣기

### 책 대여하기
1. 대여 가능한 책 카드에서 "대여하기" 버튼 클릭
2. 대여자 이름 입력
3. "대여하기" 버튼 클릭
4. 자동으로 반납 예정일이 14일 후로 설정됨

### 책 반납하기
1. 대여 중인 책 카드에서 "반납하기" 버튼 클릭
2. 즉시 반납 처리됨

### 책 검색 및 필터링
- 상단 필터 버튼으로 전체/대여가능/대여중 필터링
- 검색창에서 책 제목 또는 저자로 검색

## 프로젝트 구조

```

├── library_app.py              # Flask 애플리케이션
├── library.db                  # SQLite 데이터베이스 (자동 생성)

│   ├── library_index.html     # 메인 페이지
```

## 주의사항

- 학습/테스트 목적의 프로젝트입니다
- 실제 운영 환경에서는 사용자 인증, 권한 관리 등 추가 보안 기능이 필요합니다
